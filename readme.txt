dmd Pages provides fast edit of pages.

== Description ==

dmd Pages is a small and simple plugin to edit pages very fast.
You don't have to go into the dashboard=>pages and search your pages you want to edit.
A user can choose any published page in the adminbar.

== Installation ==

Only install and the new menu will appear in the adminbar.
